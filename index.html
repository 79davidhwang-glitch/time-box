☁️ 최초 1회 설정 — Google Drive 연동

Client ID를 한 번만 입력하면 이후 이 기기에서는 자동으로 Google Drive와 연결됩니다.
PC와 모바일 각각 한 번씩 설정하면 데이터가 완전히 동기화됩니다.

1
Google Cloud Console 설정 확인
승인된 JS 원본에 https://79davidhwang-glitch.github.io 가 등록되어 있어야 합니다.
※ 스코프: drive.file — 앱이 만든 파일만 접근 (이전 drive.appdata와 다름)
2
Client ID 입력 후 저장
이 기기에 영구 저장되며 다음부터 자동 연결됩니다.
