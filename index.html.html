☁️ Google Drive 연동 — PC · 모바일 데이터 공유
✓ 연결됨 — 데이터가 Google Drive에 자동 저장됩니다.
파일: timeflow-data.json
마지막 동기화: —

Google Drive에 연결하면 PC와 모바일에서 같은 데이터를 실시간으로 공유할 수 있습니다.

1
Google Cloud Console에서 프로젝트 생성
console.cloud.google.com 접속 → 새 프로젝트 → Google Drive API 활성화
2
OAuth 2.0 클라이언트 ID 생성
API 및 서비스 → 사용자 인증 정보 → OAuth 2.0 클라이언트 ID 만들기
유형: 웹 애플리케이션
승인된 JS 원본: 이 파일을 여는 주소 (예: http://localhost 또는 실제 도메인)
※ 로컬 파일(file://)에서 열 경우: Google Cloud Console의 테스트 사용자에 본인 계정 추가 필요
3
Client ID 입력 후 연결
발급받은 클라이언트 ID를 입력하고 Google 계정으로 로그인하세요.
동기화 설정
⚡
자동 저장 (변경 즉시 Drive에 저장)
📥
앱 열 때 Drive에서 불러오기
💡 사용 방법
1. 위에서 Google 계정으로 연결
2. 한 기기에서 할 일 등록/수정
3. 다른 기기(PC 또는 모바일)에서 이 HTML 파일 열기
4. 같은 Google 계정으로 연결하면 데이터 자동 동기화
※ 데이터는 Google Drive 앱 전용 폴더에 저장되며 다른 앱에서는 볼 수 없습니다
